# Cynthia Ramalho - Website

Bem-vindo ao repositório do site **Cynthia Ramalho**. Este projeto foi criado para exibir os serviços da empresa e disponibilizar informações de forma moderna e personalizada, utilizando tecnologias modernas para garantir alta performance e uma excelente experiência de usuário.

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido com as seguintes ferramentas:

- **[React](https://reactjs.org/)** - Biblioteca JavaScript para construção de interfaces.
- **[Vite](https://vitejs.dev/)** - Ferramenta de build extremamente rápida.
- **[Tailwind CSS](https://tailwindcss.com/)** - Framework CSS para estilização ágil e responsiva.

## 💻 Pré-requisitos

Antes de iniciar, certifique-se de que você possui o ambiente configurado com:
- [Node.js](https://nodejs.org/) (versão 18+ recomendada)
- Gerenciador de pacotes (npm, yarn, pnpm ou bun)

## 🛠️ Como executar o projeto localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/cynthia-ramalho.git
   ```

2. Acesse o diretório do projeto:
   ```bash
   cd cynthia-ramalho
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```
   *(ou `yarn install`, `pnpm install` conforme sua preferência)*

4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

5. O site estará disponível no seu navegador, geralmente no endereço [http://localhost:5173](http://localhost:5173).
